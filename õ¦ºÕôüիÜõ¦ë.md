# 八字中式心理 每日情绪引导 · 微信小程序

## 产品定位
输入出生时间 → 每天早上手机上看一张卡片（情绪+时辰+穿衣+推荐+短句收尾）。
八字推理全藏在后台，用户看到的是大白话。不走玄学，不说教，不诊断。

## 技术平台
微信小程序 · 云开发

| 项目 | 说明 |
|------|------|
| 注册 | 个人主体，免费，需身份证+手机号+邮箱 |
| ICP备案 | 上线前必须完成，约7-20工作日 |
| 类目 | 选"工具-效率"避开健康类目限制 |
| 云开发 | 上线后19.9元/月。前6个月可申请成长计划免费 |
| Open-Meteo | 天气数据，免费，无密钥 |

## 核心功能
1. 填写出生时间+性别 → 八字排盘 + 体质基线
2. 体质问卷（可选，28题，基于国家标准）→ 完成后推荐更准
3. 每日一张卡片（一屏，不滑动）

### 卡片信息构成

| 信息 | 来源 | 变动 |
|------|------|------|
| 日期+天气 | 云函数调 Open-Meteo | 每天变 |
| 情绪描述 | 八字七层规则 → 大白话翻译（见情绪规则映射表） | 每天变 |
| 时辰建议 | 子午流注 + 当日情绪 | 每天变 |
| 穿衣 | 气温 + 体质寒热 | 每天变 |
| 推荐 | 季节 + 天气 + 体质 → 一句 | 每天变 |
| 短句收尾 | 古文 + 现代翻译 + 拉回今天 | 每天变 |

## 输出话术规范
- 说人话，像朋友早上发的消息
- 不用八字术语，不用比喻，不造意象
- 直接、实在、接地气
- 禁止诊断、禁止算命话术、禁止吓唬

## 体质问卷（可选）
- 基于《中医体质分类与判定》国家标准（2009）
- 9种体质 × 28题，1-5分滑动打分
- 可选入口，不强制
- 代码可复用吾乡帖 quiz.js

## 产品理念
把养生从模糊概念变成可执行的每日日程。

## 当前范围
单人使用，不做多用户系统。

## 不做的
- 宠物系统、emoji反馈、倾诉引导、用户校准
- 功法推荐、穴位推荐、食物提示、食谱
- 天气预警、药膳推荐
- 付费墙、多用户
- AI对话/问答
