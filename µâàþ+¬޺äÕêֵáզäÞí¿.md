## 三、十神流日情绪（扩展版）

### 判定方法（按优先级）

```
第一步：日干 vs 流日天干 → 主十神
第二步：日干 vs 流日地支藏干（取主气）→ 辅十神
第三步：流日地支 vs 八字四个地支 → 刑冲合害（见第四节）
```

### 主十神 当天情绪

| 十神 | 气氛 | 情绪倾向 | 例句（柔和版） |

### 辅十神 调节系数

辅十神对主十神起调节作用。如果主十神是压力型（七杀/正官），辅十神是泄型（食神/伤官），结果会缓解。如果主辅都是压力型，会叠加。

| 组合类型 | 主+辅关系 | 调节 |
|----------|----------|------|
| 压力叠加 | 七杀+正官、七杀+七杀 | 压力×1.5，例句加安抚 |
| 压力+宣泄 | 七杀+食神、正官+伤官 | 压力×0.7，有出口 |
| 放松叠加 | 食神+偏财、比肩+食神 | 轻松×1.5，适合享受 |
| 放松+约束 | 食神+正官、偏财+七杀 | 轻松×0.7，不完全松懈 |
| 独处叠加 | 偏印+偏印、偏印+正印 | 孤僻×1.5，强制社交提示 |

## 四、十神旺衰权重

同一十神，在不同月份力度不同。旺相的十神情绪影响更大，休囚的十神情绪偏轻。

### 权重映射

| 气数 | 等级 | 情绪权重 |
|------|------|---------|
| 旺/禄 | 最强 | ×1.5 |
| 相 | 较强 | ×1.2 |
| 余/冠 | 中等 | ×1.0 |
| 休 | 较弱 | ×0.7 |
| 囚 | 弱 | ×0.5 |

**使用：** 主十神的五行 → 查当前月令的气数等级 → 确定权重 → 调节最终情绪强度。

例如：主十神是正官（属金），当前是午月（火旺克金，金处"死"地），正官的约束感大幅降低，当天的责任感/压力感偏轻。

## 五、地支刑冲合害（流日 vs 八字四柱）

流日地支与八字年、月、日、时四支发生作用。这是决定当天情绪波动大小的关键层。

### 冲（六冲）— 变动、不安、冲突

子午冲 / 丑未冲 / 寅申冲 / 卯酉冲 / 辰戌冲 / 巳亥冲

| 被冲柱位 | 影响领域 | 情绪表现 |
|----------|---------|---------|
| 年支被冲 | 大环境不安 | 隐隐不安，说不上来哪里不对 |
| 月支被冲 | 工作/人际动荡 | 工作上的情绪波动 |
| 日支被冲 | **内心被冲** | **情绪波动最大的日子，易烦躁、坐不住** |
| 时支被冲 | 小事不断 | 琐事烦心，小情绪累积 |

> 特别关注：日支被冲 = 当天情绪波动加倍。例句：「今天心里不太平，静不下来也不用强压，换个地儿坐坐。」

## 十二、输出话术规范

### 核心原则

说人话。像朋友早上发给你的一条消息。不用八字术语，不用西方比喻，不造意象，不抒情。**直接、实在、接地气。**

### 情绪描述

- 把八字逻辑翻译成大白话，用户不需要知道"正印日""丑未冲"是什么
- 不绕弯子，三句以内把今天的感觉说清楚
- 不用"像是""像"，直接说"今天心里可能……""今天容易……"

**翻译对照：**

| 八字逻辑 | 大白话 |
|----------|--------|
| 正印日 | 今天心里比较静 / 适合一个人待着 / 不急不赶 |
| 七杀日 | 今天容易有人跟你过不去 / 有压力 |
| 伤官日 | 想法多、嘴快 / 容易说错话 |
| 食神日 | 今天轻松、适合吃好的 |
| 偏印日 | 想一个人待着 / 别逼自己社交 |
| 日支被冲 | 心里不太平 / 坐不住 / 有点烦躁 |
| 日支逢合 | 心里顺畅 / 有被接住的感觉 |
| 自刑 | 自己跟自己较劲 / 对自己温柔点 |
| 丑未冲 | 说不清的别扭 / 不要细想 |
| 空亡 | 做什么都觉得差口气 / 不是你的问题 |
| 木旺怒 | 今天容易急 / 少生气 |
| 金旺悲 | 容易感伤 / 出门走走 |
| 火旺亢 | 停不下来 / 给自己踩个刹车 |
| 土旺思 | 脑子转太多圈 / 不如动手做一件 |
| 水旺恐 | 容易想太多不好的 / 只看眼前的事 |

### 时辰建议

根据子午流注+流日情绪，推一个最重要的时段，用大白话：
- "中午11点到1点精神最好，重要的事放这个时段"
- "下午3点到5点容易犯困，别排太费脑的事"

### 穿衣+推荐

- 穿衣：气温+体感，一句大白话。"广州闷热，薄衫透气"
- 推荐：季节+天气+体质，一句饮食/生活小建议

### 短句收尾

格式固定：**古文 → 翻译 → 拉回今天**

```
物来则应，过去不留。
东西来了就接住，过去了就松手。
今天这点不踏实也是，来了就来了，别攥着不放。
```

## 十四、边界规则

以下**绝对不输出**：
- 暗示厄运：不出现「灾」「祸」「凶」「劫数」「倒霉」「血光」

- 算命话术：不说「桃花」「破财」「官非」
- 自杀/自伤暗示：任何相关话题不接
